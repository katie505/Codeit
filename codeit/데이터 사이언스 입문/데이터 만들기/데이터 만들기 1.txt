<웹 페이지 가져오기>
웹 페이지 구성
*데이터를 모으기 위해서 '태그의 구조'는 중요함

1) HTML 태그의 구성 
- 태그 이름 : <>기호 안에 태그 이름을 넣음
- 속성 : 일반적으로 속성 이름과 속성 값을 하나의 쌍으로 갖게 됨.
- 한 태그가 여러 속성을 가질 수도 있음

2) **HTML 태그의 구조
- 한 페이지의 HTML 태그는 서로 연결되어 있음 = 이 구조가 마치 가계도나 나무와 유사해서, 부모 관계 혹은 트리 구조라고도 부름
EX) <ul>태그가 <li>태그를 포함하고 있으니 <ul>태그가 부모, <li>태그가 자녀

3) DOCTYPE 선언 
- HTML 파일을 쓸 때는 가장 먼저 <!DOCTYPE> 선언을 써야 함
- 가장 최신 버전인 HTML 5를 사용하기 위해서는 이렇게만 쓰면 됨

4) Title 태그
- 페이지의 제목은 <title> 태그에 써주면 됨
- 브라우저의 탭이나 방문 기로겡 나와 있는 제목

5) h1 ~ h6 태그
- 가장 중요한 머리말은 <h1> 그 다음으로 중요한 머리말은 <h2> ~
- 각 머리말의 크기는 나중에 마음대로 설정할 수 있지만, 따로 설정해주지 않으면 <h1>부터 순서대로 작아짐

6) p 태그
- 보통 문단은 <p> 태그 안에 넣음
- <p>태그 위, 아래에는 기본적으로 여백이 조금씩 있음

*태그에 대해 더 알고싶다면 'https://www.w3schools.com/tags/ref_byfunc.asp'

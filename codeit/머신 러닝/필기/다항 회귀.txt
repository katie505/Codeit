<다중 회귀>
- 함수의 차수가 더 높을수록 더 굴곡이 많은 곡선. 데이터를 보고 적합할 것 같은 모양을 골라야 함
- 입력 변수를 제곱하고 세제곱한 값들로 가상의 열을 추가하고, 이 새로운 열들을 마치 입력 변수처럼 취급
=> 다중 선형 회귀랑 똑같은 방식으로 하면 됨

1. 속성이 하나인 경우(단일 속성 다항 회귀)
2. 속성이 많은 경우(다중 다항 회귀)

*선형 회귀는 학습 데이터에 가장 잘 맞는 일차식을 찾아서 예측을 하는 알고리즘
<-> 다항 회귀는 고차식을 찾아서 예측하는 알고리즘

속성 사이의 복잡한 관계
- 단순 선형 회귀를 사용하면 두 변수가 서로 독립적
=> 단순 선형 회귀를 쓰면 속성 하나하나가 다 독립적이기 때문에 이런 변수들 사이의 관계를 최종 예측 결과에 반영하지 못함
- 속성들을 서로 곱해서 차항을 높여주면, 그러니까 선형 회귀 문제를 다항 회귀 문제로 만들어주면 속성들 사이에 있을 수 있는 복잡한 관계들을 프로그램에 학습시킬 수 있음
